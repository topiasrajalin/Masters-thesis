# Tests Directory

Unit and integration tests for Python ETL scripts and analysis code.

## Structure

Place test files here to validate:
- Data loading and preprocessing
- Currency conversions
- Return calculations
- Data integrity checks

## Running Tests

```bash
python -m pytest tests/
```

## Example Test Categories

- `test_etl_*.py` - Tests for ETL data processing scripts
- `test_calculations_*.py` - Tests for calculation accuracy
- `test_config_*.py` - Tests for configuration handling
